# Smart-IOT-Weather-Monitoring-System
A smart IoT-based weather monitoring system using ESP32 and sensors (DHT22, BMP280) to collect real-time data (temperature, humidity, pressure) and send it over WiFi to Blynk and ThingSpeak for visualization, alerts, and analytics. The circuit is Validated via Wokwi simulation and designed using EasyEDA for real world hardware use.
